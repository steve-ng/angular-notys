showNoty API Guide

To call, simply inject noty into your angular controller/service/factory etc... An example as shown below

noty.showNoty({
  text: "Deleted an email",
  ttl: 1000000, //time to live in miliseconds
  type: "default", //success, warning 
  options: ['Undo', 'Dismiss'], 
  optionsCallBack:  function callback(optionClicked, optionIndexClicked) {
    //handling code for options clicked
  }
})

A list of all the possible options

text - Message to show in the notification
ttl - time to live in miliseconds before the notification disappearing on its own
type -theme of the message, can be default, success or warning
options (optional) - List of options to provide in the notification
optionsCallBack (optional) - if options is stated and when user clicked on one of the options, this callback will be called.
